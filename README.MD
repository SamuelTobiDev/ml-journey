## Projects

### Project 1 — Titanic Survival Predictor
Predicts passenger survival using classical ML models trained on the Titanic dataset.

| Model | Accuracy | Precision | Recall | F1 |
|---|---|---|---|---|
| Logistic Regression | 81.0% | 77.8% | 71.0% | 74.2% |
| Decision Tree | 83.8% | 80.3% | 76.8% | 78.5% |
| Random Forest | 84.4% | 82.5% | 75.4% | 78.8% |
| XGBoost | 82.1% | 78.5% | 73.9% | 76.1% |
| XGBoost tuned | 78.2% | 73.4% | 68.1% | 70.7% |
